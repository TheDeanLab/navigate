class aslm.model.features.image_writer.ImageWriter(model, data_buffer=None, sub_dir='', image_name=None)

Bases: object

Class for saving acquired data to disk.

Parameters:

  • model (aslm.model.model.Model) – ASLM Model class for controlling hardware/acquisition.

  • data_buffer ([SharedNDArray]) – data_buffer will use model’s default data_buffer if it’s not specified

  • sub_dir (str) – Sub-directory of self.model.configuration[‘experiment’] [‘Saving’][‘save_directory’] indicating where to save data

  • image_writer (str) – Optionally override the generate_image_name() naming scheme.

Return type:

None
