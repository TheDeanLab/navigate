navigate.model.devices.stages.stage_galvo.GalvoNIStage

class navigate.model.devices.stages.stage_galvo.GalvoNIStage(microscope_name, device_connection, configuration, device_id=0)

Bases: StageBase

Galvo Stage Class (only supports one axis)

Generic analog controlled stage. Could be used to control piezos, galvos, etc. Currently set up to handle National Instruments data acquisition cards.

Retrieves the volts per micron from the configuration file and uses that to determine the correct voltage to send to the stage.
