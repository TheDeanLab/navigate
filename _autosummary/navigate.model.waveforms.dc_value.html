navigate.model.waveforms.dc_value

navigate.model.waveforms.dc_value(sample_rate=100000, sweep_time=0.4, amplitude=1)

Returns a numpy array with a DC value Used for creating the resonant galvo drive voltage.

Parameters:

  • sample_rate (Integer) – Unit - Hz

  • sweep_time (Float) – Unit - Seconds

  • amplitude (Float) – Unit - Volts

Returns:

waveform

Return type:

np.array

Examples

>>> typical_galvo = dc_value(sample_rate, sweep_time, 1)
